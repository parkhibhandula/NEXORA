VIBE COMMERCE
A full-stack shopping cart built for Vibe Commerce with product listing, cart management, quantity updates, and mock checkout.

#TECH STACK used ReactJS,NodeJS,ExpressJS,MongoDB,Axios,CORS 

#FEATURES
-Responsive design
-Product listing (DB stored)
-Add to cart
-Increase/Decrease quantity
-Remove Item from cart
-Automatic amount calculation
-Mock checkout

RUN BACKEND

Setup-

git clone https://github.com/parkhibhandula/NEXORA.git


cd backend
node app.js

RUN FRONTEND

cd frontend
npm run dev# NEXORA
